🌐 Personal Portfolio Website

This project is a modern and responsive personal portfolio website designed to showcase my skills, projects, certifications, and professional journey. It serves as a digital representation of my work, highlighting my passion for Data Analytics, continuous learning, and problem-solving.

🚀 Features
🏠 Clean and attractive landing page
👨‍💻 About Me section introducing my background and career aspirations
🛠️ Skills section showcasing technical competencies
📂 Projects gallery with detailed project highlights
📜 Certifications and achievements display
📱 Fully responsive design for desktop and mobile devices
📞 Contact section for professional networking opportunities
💻 Technologies Used
HTML5
CSS3
JavaScript
Git & GitHub Pages
🎯 Purpose of This Project

The goal of this portfolio is to create a centralized platform where recruiters, hiring managers, and fellow professionals can explore my work and track my growth as an aspiring Data Analyst. This project reflects my commitment to learning, building impactful projects, and continuously improving my technical skills.
